## 🔧 Технические детали

### Изменения в `index.html`

#### 1. **Обработка полей в начале записи**
```javascript
// Parse fields at the start (before any section)
if (!currentSection && line.match(/^\*\*(План|Выполнено|RPE|Место|Время|Локация|Тип):/)) {
    if (line.match(/^\*\*RPE:/)) {
        currentSession.rpe = line.replace(/^\*\*RPE:\*\*\s*/, '').trim();
    }
    // ... и т.д.
}
```

#### 2. **Гибкие regex для секций**
```javascript
// Section headers (support both **, ####, and ### formats)
if (line.startsWith('**До тренировки:**') || 
    line.match(/^(###|####)\s*(⚙️\s*)?(Параметры тренировки|Параметры|Состояние перед тренировкой|До тренировки)/i)) {
    currentSection = 'before';
    continue;
}
```

#### 3. **Гибкие regex для полей**
```javascript
if (line.match(/^-\s*\*?\*?Сон:?\*?\*?/i)) {
    currentSession.sleep = line.replace(/^-\s*\*?\*?Сон:?\*?\*?\s*/i, '').trim();
}
```

Объяснение regex:
- `\*?\*?` - опциональные звездочки (болд)
- `i` флаг - регистронезависимый поиск
- Альтернативные названия через `|`

#### 4. **Улучшенное извлечение RPE и длительности**
```javascript
// Text format: "RPE: 8/10" or "ощущения 8/10"
const rpeMatch = line.match(/(?:RPE|ощущения)[:\s]*(\d+\/10)/i);
if (rpeMatch) {
    currentSession.rpe = rpeMatch[1];
}
```
