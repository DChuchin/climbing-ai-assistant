### 4. **Улучшен парсер (index.html)**

Добавлена логика извлечения RPE и длительности из JSON блоков:
```javascript
// Extract RPE and duration from JSON blocks if not found in **Итоги:**
if (line.includes('"rpe"') && !currentSession.rpe) {
    const match = line.match(/"rpe"\s*:\s*(\d+)/);
    if (match) {
        currentSession.rpe = match[1] + '/10';
    }
}
if (line.includes('"duration_min"') && !currentSession.duration) {
    const match = line.match(/"duration_min"\s*:\s*(\d+)/);
    if (match) {
        currentSession.duration = match[1] + ' мин';
    }
}
```
