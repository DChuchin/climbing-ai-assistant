# 📋 ФОРМАТ ЗАПИСИ ТРЕНИРОВКИ

## ✅ ЕДИНЫЙ ФОРМАТ (используй для всех новых тренировок!)

```markdown
## 📅 YYYY-MM-DD | День недели | Тип тренировки

**До тренировки:**
- Сон: X часов (качество Y/10)
- Восстановление: X/10
- Состояние: [описание]
- Вес: XX кг

**Выполненная работа:**

[Детали по блокам:]

- Разминка: XX мин
- Лазание: YY проблем (детали)
- ОФП/Сила: [упражнения, веса, подходы]
- Заминка: XX мин

**Итоги:**
- RPE: X/10
- Длительность: XX мин
- Самочувствие после: X/10

**Заметки:**
- [Что получилось хорошо]
- [Что было сложно]
- [Боли/дискомфорт]
- [Технические находки]

---
```

## 🎯 ПОЧЕМУ ИМЕННО ЭТОТ ФОРМАТ?

### 1. **Заголовок `## 📅 YYYY-MM-DD | День | Тип`**
   - ✅ Дата в ISO формате → легко парсить
   - ✅ Всегда начинается с `## 📅` → однозначный маркер
   - ✅ Вся ключевая информация в одной строке
   - ✅ Можно легко найти и сортировать

### 2. **Четкие секции**
   - `**До тренировки:**` - состояние перед
   - `**Выполненная работа:**` - что сделал
   - `**Итоги:**` - RPE, длительность
   - `**Заметки:**` - наблюдения

### 3. **Простой парсер**
```javascript
// Ищем только строки, начинающиеся с ## 📅
if (line.startsWith('## 📅')) {
    // Парсим дату из формата YYYY-MM-DD
    const match = line.match(/## 📅 (\d{4}-\d{2}-\d{2})/);
    date = match[1];  // Готово!
}
```

### 4. **Легко читать человеку**
   - Дата всегда на одном месте
   - Структура предсказуема
   - Можно быстро найти нужную тренировку

## 📝 ПРИМЕРЫ

### Пример 1: Силовая тренировка

```markdown
## 📅 2026-02-13 | Пятница | Интенсивность + Проекты

**До тренировки:**
- Сон: 8 часов (качество 7/10)
- Восстановление: 7/10
- Состояние: Готов к интенсивной работе
- Вес: 85 кг

**Выполненная работа:**

Разминка (15 мин):
- Суставная гимнастика
- Висы, легкие подтягивания

Подтягивания с весом (15 мин):
- Разминка: +10кг × 4, +20кг × 4, +30кг × 3
- Рабочие: +35кг 4 × 3
- Отдых: 3 мин

Блокировки (15 мин):
- 90°, 120°, 150°: 3 × 3 по 10 сек
- Отдых: 3 мин

Кампусборд (15 мин):
- Лесенка 1-10 вверх-вниз
- 3 подхода
- Отдых: 5 мин

Проекты (40 мин):
- 6C+: 3 проблемы (2 залез, 1 в работе)
- 7A: 2 попытки (не залез)
- Отдых: 7-8 мин

Кор (10 мин):
- L-seat: 3 × 45 сек
- Планка: 3 × 45 сек

**Итоги:**
- RPE: 8/10
- Длительность: 110 мин
- Самочувствие после: 7/10 (приятная усталость)

**Заметки:**
- Подтягивания +35кг идут стабильно
- Блокировки держу 10 сек уверенно
- На проектах 6C+ прогресс - 2 залез!
- 7A пока сложно, но чувствую что близко
- Пальцы в порядке, без перегрузки

---
```

## 🔧 КАК ПЕРЕФОРМАТИРОВАТЬ СТАРЫЕ ЗАПИСИ?

**Не обязательно переформатировать все!** Парсер будет работать со старыми форматами.

Но если хочешь привести к единому виду:
1. Найди заголовок тренировки (любой формат)
2. Замени на `## 📅 YYYY-MM-DD | День | Тип`
3. Проверь что есть все 4 секции
4. Готово!

## 💡 AI БУДЕТ АВТОМАТИЧЕСКИ ИСПОЛЬЗОВАТЬ ЭТОТ ФОРМАТ

При записи новых тренировок в `progress_log.md` всегда используется этот формат.
